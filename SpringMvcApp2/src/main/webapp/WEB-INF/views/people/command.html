Use only Get calls.

/people/{id} - show one person. (GET)

/people/new - create new person. (GET)

/people/{id}/edit - change person. (GET)

/people - show all person. (GET)

/people - create new person. Call from "new.html". (POST)

/people/{id} - update one person. Call from "edit.html". (PATCH)

/people/{id} - delete one person. Call from "show.html". (DELETE)
